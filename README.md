## About

Simple shop-page for Arthrohard using html, sass, pure js. Build with Vite.
- Live version: https://transcendent-pasca-c48910.netlify.app/

## How to run

```bash
git clone https://github.com/rafal19987/shop-page.git
cd shop-page
npm i
npm run dev
```
